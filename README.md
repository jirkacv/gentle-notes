# GentleNotes

GentleNotes is an interview project about simple note management


## API e2e test
Switch to api folder and run
``` npm i && npm run test:e2e ```


## Run API
Switch to api folder and run
``` npm i && npm start ```


## Run UI
Switch to ui folder and run
``` npm i && npm start ```

## Accessing the app
Go to http://localhost:3000
